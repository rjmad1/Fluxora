# UI Design References
## Fluxora: Social Media Blast Mockups

This document serves as the design reference for the user stories and PRD, illustrating the premium glassmorphism interface and visual structure of Fluxora.

### US-02 / US-04: Unified Composer & Scheduling Dashboard

Below is the design mockup representing the Unified Composer interface, network previews, and scheduler layout.

![Fluxora Dashboard Mockup](file:///C:/Users/rajaj/.gemini/antigravity-ide/brain/9c7c7cab-2e0a-44ba-b7aa-20a8f7c654aa/fluxora_dashboard_mockup_1781482002566.png)

* **Key Visual Elements**:
  - Left navigation pane: Workspace switcher, dashboard, calendar scheduler, asset library, presets, and integration settings.
  - Center panel: Rich text composer supporting hashtags, media attachments, and scheduling configurations.
  - Right panel: Dynamic preview cards showcasing how the post will look on Meta Instagram, X (Twitter), and LinkedIn.
  - UI Style: HSL tailored deep slate backgrounds, glassmorphic card boundaries, and vibrant purple-to-blue gradient highlights.
